### 7.1 사업비 산정 — 표준 원가 구조 (★)

#### 7.1.1 정보화사업 표준 원가 구성
한국SW산업협회 + 행안부 정보화사업 가이드:
```
사업비 = 직접인건비 + 직접경비 + 제경비 + 기술료
       (+ 부가세 10%)
```

#### 7.1.2 직접인건비 산정 (★ 가장 큰 비중 70~80%)
**한국SW산업협회 SW기술자 노임단가 적용** (매년 갱신, 2024 기준):
```
| 등급 | 일 노임단가 | M/M 단가 (20일) |
|------|-----------|---------------|
| 특급기술자 | 약 47만원 | 약 940만원 |
| 고급기술자 | 약 38만원 | 약 760만원 |
| 중급기술자 | 약 30만원 | 약 600만원 |
| 초급기술자 | 약 22만원 | 약 440만원 |
| 고급기능사 | 약 18만원 | 약 360만원 |
| 중급기능사 | 약 14만원 | 약 280만원 |
| 초급기능사 | 약 11만원 | 약 220만원 |
```

**산정 방식**:
- 등급별 인원 × M/M × 노임단가
- 본 사업 도메인별 적정 인원·M/M 도출 (Phase 4 인력 투입 계획 연계)

#### 7.1.3 기능점수(FP) 기반 산정 (대형 SI 50억+)
**NIA 기능점수 가이드** — 행정·공공 정보화 기준:
- 단가: **약 51만원/FP** (2024 기준)
- 산정: ILF / EIF / EI / EO / EQ 기능점수 합산
- 행안부 정보화사업 가이드 권고 (개발비 산정)

#### 7.1.4 직접경비
| 항목 | 비율(평균) | 설명 |
|------|----------|------|
| 출장비 | 1~2% | 발주처 방문, 회의 |
| HW 도입비 | 변동 | 서버·스토리지·네트워크 |
| SW 라이선스 | 변동 | DB·미들웨어·솔루션 |
| 통신비 | 1% | 회선·인터넷 |
| 보안 도구 | 변동 | EDR·SIEM·암호 |
| 소모품 | 0.5% | 일반 소모품 |

#### 7.1.5 제경비 (간접비)
- **공식**: 직접인건비 × 110~120%
- **포함**: 사무실 임차·관리비·복리후생·교육·연수
- **비율 기준**: 한국엔지니어링협회 기술용역 대가 기준

#### 7.1.6 기술료 (이윤·기술 가치)
- **공식**: (직접인건비 + 제경비) × 20~40%
- **결정 기준**:
  - 사업 규모 / 위험도 / 기술 난이도 / 국산 솔루션 비중
  - 통상 25~30% 적용

#### 7.1.7 부가가치세
- 사업비 합계 × 10%
- 일부 면세 항목 제외

### 7.3 가격경쟁력 입증 (★ 가격평가 점수 직결)

#### 7.3.1 추정가 대비 입찰가 비율
- **공식**: 점수 = 만점 × (최저가 / 입찰가) × 보정계수
- **저가심사 기준**: 추정가의 **80% 미만** 시 적정성 심사
- **권장 비율**: 추정가의 **85~95%** (저가심사 회피 + 가격 점수 확보)

#### 7.3.2 단가 비교 (필요 시)
- 동종 사업 평균 단가 대비
- 자사 과거 실적 단가 대비
- → "합리적 가격" 입증

### 7.4 도메인별 가격·사업비 특화

#### 7.4.1 빅데이터 신규 구축
- **인건비 비중**: 60~70%
- **HW/SW**: 20~30% (서버·DB·BI·라이선스)
- **클라우드 사용 시**: 운영비 별도 산정 (월 단가 × 기간)

#### 7.4.2 빅데이터 유지보수·운영 ★
- **연간 운영비 = 도입비 × 10~12%** (행안부 고시)
- **인건비 비중**: 70~80% (상주 인력)
- **HW 유지보수**: 도입 HW × 8~10%
- **SW 유지보수**: 도입 SW × 15~22%
- **장기계약 할인**: 3년 계약 시 5~10% 인하 가능

#### 7.4.3 빅데이터 분석
- **단기 사업** (3~12개월) 단가 1~10억
- **인건비 비중**: 70~80%
- **분석 인프라**: 10~20% (GPU·SW)
- **외부 데이터 구매**: 5~10% (필요 시)

#### 7.4.4 빅데이터 고도화·차세대
- **이중 운영 비용** (병행운영 기간 가중)
- **데이터 이관 비용** (검증·동기화)
- **롤백 대비 인프라** (TO-BE 구축 + AS-IS 유지)
- → 신규보다 통상 1.2~1.5배

#### 7.4.5 AI 사업
- **GPU 인프라** 비중 ↑ (자체 또는 클라우드)
- **학습 데이터 구축** (라벨링 아웃소싱)
- **모델 평가·검증** (TTA Lv.4 등)

#### 7.4.6 인프라·클라우드 전환
- **HW/SW** 비중 ↑ (50%+)
- **인건비** 30~40%
- **CSAP 인증 비용** 별도

#### 7.4.7 CCTV·통합관제
- **카메라·전송망 HW** 비중 매우 큼 (40~50%)
- **AI 분석 SW** 라이선스
- **통합관제센터 시설** (모니터월·콘솔)
- **5대 안전망 연계 모듈**

#### 7.4.8 데이터 거버넌스
- **인건비 비중** 70~80% (컨설팅성)
- **메타데이터·DQM 솔루션** 라이선스
- **KCMVP 검증 모듈**

### 7.5 정량 ROI · 시민·정책 효익 (★공공 차별화)

마케팅 사업과 달리 공공입찰은 **"국민·정책 효익"**을 ROI로 환산:

#### 7.5.1 직접 ROI (정량)
- **TCO 절감**: 5년 누계 (인건비·라이선스·인프라 절감)
  ```
  As-Is 5년 운영비: ?? 원
  TO-BE 5년 운영비: ?? 원
  절감 효과: ?? 원 (??%)
  ```
- **자동화 효과**: 인력 시간 × 시급 × 인원 × 연간
- **장애 감소**: 다운타임 × 비즈니스 손실/시간

#### 7.5.2 간접 효과 (정성·정량)
- **시민·국민 효익**:
  - 처리 시간 단축 (분/시간 → 초/분)
  - 24/7 서비스 (휴일 대응)
  - 디지털 격차 해소
- **공무원 업무 효율**:
  - 보고서 작성 시간 절감
  - 데이터 기반 의사결정
- **정책 효과**:
  - 정책 집행 속도·정확도 향상
  - 데이터 기반 정책 평가

#### 7.5.3 도메인별 효익 사례
- **빅데이터 분석**: 정책 N건 채택 + 예산 절감 ?? 억
- **AI**: 민원 자동 분류로 처리시간 80% 단축
- **인프라**: 시스템 응답 N배 개선 + 5년 TCO 30% 절감
- **CCTV**: 범죄 발생률 ??% 감소 + 신고 응답 ??% 단축
- **거버넌스**: 데이터 활용 N배 + 침해사고 0건

#### 7.5.4 정책 부합성 효과
- 디지털플랫폼정부 실현 기여
- 데이터3법 활용 확대
- 국정과제 ?? 직접 기여
- → "이 사업이 정부 정책 실현의 한 축"

### 7.6 위험·잠재 비용 (선제적 안전 장치)
- **잠재 위험** 식별 + 대응 비용 별도 명시 (무리한 저가 입찰 회피)
- **변경 관리 예비비** (5%)
- **유지보수 인수 비용** (사업 종료 후)
