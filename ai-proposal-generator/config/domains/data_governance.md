## 데이터 품질(DQ) 6대 지표
1. **정확성 (Accuracy)** — 실제 값과의 일치
2. **완전성 (Completeness)** — 결측치 비율
3. **일관성 (Consistency)** — 시스템 간 동일 값
4. **적시성 (Timeliness)** — 최신성
5. **유효성 (Validity)** — 형식·도메인 적합
6. **유일성 (Uniqueness)** — 중복 없음

## 비식별화 기법 (가명정보 처리)
| 기법 | 설명 | 적용 대상 |
|------|------|----------|
| 가명처리 (Pseudonymization) | 식별자 → 가명 (복원 가능) | 이름·주민번호·계정 |
| 총계처리 (Aggregation) | 합계·평균으로 변환 | 통계 분석 |
| 데이터 삭제 (Suppression) | 식별자 제거 | 직접식별자 |
| 범주화 (Categorization) | 구간화 | 나이·소득 |
| 마스킹 (Masking) | 일부 가림 (예: ***-**-1234) | 화면 표시 |
| k-익명성 / l-다양성 / t-근접성 | 그룹 단위 보호 | 통계 공개 |
| 차분 프라이버시 (DP) | 노이즈 추가 | AI 학습 데이터 |
