# 공공입찰 평가기준 가이드 (나라장터)

> 나라장터(국가종합전자조달시스템) 공공입찰 평가체계 정리.
> Phase 1(SUMMARY), Phase 2(INSIGHT), Phase 6(WHY US), Phase 7(INVESTMENT)에서 참조.

## 입찰 방식 4유형

### 1. 협상에 의한 계약 (가장 일반)
- **근거**: 국가계약법 시행령 제43조 / 지방계약법 시행령 제43조
- **특징**: 가격 + 기술 종합 평가, 우선협상자 선정 후 협상
- **표준 배점**: 기술 80점 + 가격 20점 (또는 90:10, 70:30)
- **적용**: 정보화·SI·컨설팅 등 기술 비중 높은 사업 — IT·AI·빅데이터·CCTV 대부분

### 2. 적격심사 (가격 중심 + 최소 기술 검증)
- **근거**: 국가계약법 시행령 제42조
- **특징**: 가격 입찰 + 적격심사 통과 시 낙찰
- **배점**: 가격평가 + 수행능력 + 신인도 (각 PASS/FAIL)
- **적용**: 단순·반복 사업, 표준 시스템 도입

### 3. 일반경쟁 (최저가)
- **특징**: 가격만으로 낙찰 (특수 사업 외 기피)
- **적용**: 단순 물품·용역

### 4. 수의계약 / 제한경쟁 / 지명경쟁
- **특징**: 특수 자격·실적·규모 제한
- **적용**: 보안·국방·특수 기술

## 협상에 의한 계약 — 표준 평가표 (정성+정량)

### 기술능력평가 (보통 80점)

| 대분류 | 중분류 | 표준 배점 | 평가 포커스 |
|--------|--------|----------|-----------|
| **수행 계획** | 사업 이해도 | 5~10 | RFP 요구사항 완벽 이해 + 정책 부합성 |
| | 추진 체계 | 5~10 | 조직·역할·소통 체계 |
| | 추진 일정 | 5 | WBS 현실성 + 리스크 대응 |
| **수행 방법** | 기술 적합성 | 10~15 | 아키텍처·기술스택 적정성 |
| | 분석·설계 | 5~10 | 요구사항 추적·설계 방법론 |
| | 구축·테스트 | 5~10 | 단계별 구축·시험 계획 |
| | 운영·유지보수 | 3~5 | SLA·인수인계·교육 |
| **품질·보안** | 품질관리 | 5 | 품질 지표·테스트·검증 |
| | 정보보호 | 5~10 | 망분리·암호화·접근통제·인증 |
| **수행 역량** | 유사 실적 | 10~15 | 동일·유사 사업 수행 건수·규모 |
| | 인력 보유 | 5~10 | PM·PL·기술책임자 자격·경력 |
| | 재무 안정성 | 2~3 | 신용평가·재무비율 |

### 가격평가 (보통 20점)
- **공식**: `점수 = 만점 × (최저가 / 입찰가) × 보정계수`
- **저가심사 기준**: 추정가의 80% 미만 시 적정성 심사
- **예시**: 추정가 10억 / 입찰가 8.5억 / 최저가 8억 → `20 × (8/8.5) = 18.8점`

### 신인도 (가산점, ±2~5점)
- **가산**: 정부포상·국가품질상·고용창출·일자리 협력기업·여성/장애인 기업
- **감점**: 부정당업자 제재·세금 체납·기술자료 침해

## 정성평가 채점 기준 (평가위원 관점)

| 등급 | 점수율 | 평가위원 인식 |
|------|--------|------------|
| 매우 우수 | 90~100% | RFP 100% 충족 + 차별화 + 검증된 실적 |
| 우수 | 80~90% | RFP 충족 + 일부 차별화 |
| 보통 | 70~80% | RFP 충족 (평균적) |
| 미흡 | 60~70% | RFP 일부 미충족 |
| 매우 미흡 | <60% | RFP 핵심 미충족 |

→ **수주 안전권 = 평균 90% 이상** 평가받도록 작성

## 평가위원 구성 (예시)
- 7~10명 (분야별 전문가)
- 학계 30~40% / 산업계 30% / 발주처 20% / 외부 자문 10%
- 평가 시간: **1제안서당 30~60분**
  → 핵심 메시지가 30초~1분 안에 전달되도록 설계 필수
  → Action Title + 시각화 + Win Theme의 중요성

## Win Theme — 공공입찰 특화 패턴
1. **정책 부합성** — 디지털플랫폼정부 / 스마트시티 / K-Cloud 등 국정과제 연계
2. **검증된 안정성** — 유사 사업 무중단 운영 + 정량 성과
3. **위험 최소화** — 단계적 도입 + 보안 인증 + DR

## 입찰 자격 요건 (RFP에서 자주 등장)
- **소프트웨어 사업자 신고증** (구 SW사업자) — 행안부
- **정보통신공사업** — 일정 등급(예: 2등급 이상)
- **엔지니어링 사업자** — 정보통신·정보처리
- **CMMI Lvl. 3+ 인증** — 대형 정보화 사업
- **ISO 9001 / 27001 / 27701** — 품질·보안·개인정보
- **계약이행 능력 (재무)** — 신용평가 BB+ 이상
- **유사실적** — 최근 3~5년, 계약규모 50~70% 이상
- **PM 자격** — 정보처리기술사 또는 PMP + 유사 사업 PM 경력

## 가격 산정 (Phase 7 INVESTMENT 작성 시)

### 정보화사업 표준 원가 구조
```
사업비 = 직접인건비 + 직접경비 + 제경비 + 기술료 (+ 부가세)
```

- **직접인건비**: 등급별 노임단가 × M/M
  - SW기술자 등급: 특급 / 고급 / 중급 / 초급
  - 노임단가 출처: **한국SW산업협회 SW기술자 노임단가** (매년 갱신)
  - 2024년 기준: 특급 약 47만원/일, 고급 38만원/일, 중급 30만원/일, 초급 22만원/일
- **직접경비**: 출장·SW 라이선스·HW·통신·보안 등 실비
- **제경비**: 직접인건비 × 110~120% (간접비)
- **기술료**: (직접인건비 + 제경비) × 20~40%

### 기능점수(FP) 기반 산정 (대형 SI)
- **NIA 기능점수 가이드** — 행정·공공 정보화 기준
- 단가: 약 **51만원/FP** (2024 기준, 매년 변동)

## 제안서 작성 시 평가위원 어필 포인트

✅ **첫 페이지 임팩트** — 30초 안에 Win Theme 3개 전달
✅ **시각적 도식화** — 글 < 다이어그램 < 도식 + 정량 수치
✅ **요구사항 추적 매트릭스** — 누락 없음을 증명
✅ **유사 실적 정량 성과** — 모호한 "성공" 대신 구체 수치
✅ **핵심 인력 이력서** — 본 사업과 유사한 PM 경력 강조
✅ **인증·자격 일람표** — 한 페이지에 모두 정리
✅ **위험 관리** — 위험 식별 + 완화 전략 매트릭스 (감점 회피)

## 자주 보는 감점 요인
❌ RFP 요구사항 누락 (RTM 미작성)
❌ 평가기준 미숙지 (배점 분포에 맞지 않는 비중)
❌ 정성적 성과 표현 ("우수한", "혁신적인")
❌ 유사 실적 부족 (5건 미만)
❌ 핵심 인력 부재 (PM·PL 명시 안됨)
❌ 보안 통제 누락 (망분리·접근통제·암호화)
❌ 일정 비현실성 (마일스톤 정합성 부족)
