# Impact-8 Framework 및 콘텐츠 규칙

## 제안서 구조: Impact-8 Phase

| Phase | 이름 | 비중 | 내용 |
|-------|------|------|------|
| 0 | HOOK (티저) | 3-10p (5%) | 임팩트 있는 오프닝, 핵심 메시지, 비전 |
| 1 | SUMMARY | 3-5p (5%) | Executive Summary (의사결정자용 5분 요약) |
| 2 | INSIGHT | 8-15p (10%) | 시장 환경 + 문제 정의 + 숨겨진 니즈 |
| 3 | CONCEPT & STRATEGY | 8-15p (12%) | 핵심 컨셉 + 차별화 전략 + 경쟁 우위 |
| 4 | ACTION PLAN (핵심) | 30-60p (40%) | 상세 실행 계획 + 콘텐츠 예시 + 채널별 전략 |
| 5 | MANAGEMENT | 6-12p (10%) | 조직 + 운영 + 품질관리 + 리포팅 |
| 6 | WHY US | 8-15p (12%) | 수행 역량 + 유사 실적 + 레퍼런스 |
| 7 | INVESTMENT & ROI | 4-8p (6%) | 투자 비용 + 정량적 효과 + ROI |

총 70-140p (프로젝트 규모에 따라 조정)

## Win Theme (수주 전략 메시지)

제안서 전체에 반복되는 3대 핵심 수주 전략 메시지

```python
WIN_THEMES = {
    "data": "데이터 기반 타겟 마케팅",
    "community": "시민 참여형 브랜드 빌딩",
    "integration": "온-오프라인 통합 시너지",
}
```

- 각 섹션 구분자에 관련 Win Theme 표시
- 슬라이드 내에서 Win Theme 뱃지로 강조
