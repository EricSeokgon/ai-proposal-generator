# 캠페인 정보: 크래프트볼트 (CraftVolt)

## 1. 브랜드/제품 정보

| 항목 | 내용 |
|------|------|
| 브랜드명 | 크래프트볼트 (CraftVolt) |
| 공식 홈페이지 | http://www.craftvolt.kr/ |
| 제품 카테고리 | 충전식 전동공구 (21V 리튬이온 배터리 기반) |
| 주요 제품 | 체인톱, 송풍기, 전동가위, 배터리/액세서리 |
| 제품 특징 | 브러시리스 모터, 경량 설계, 21V 고출력 |
| 브랜드 컬러 | 블랙 + 오렌지(#ef7b00) |
| A/S 정책 | 1년 이내 무상 A/S, 배터리 수리 서비스 |

### 주요 제품 라인업
- **21V 8인치 체인톱** — 가벼운 무게 + 강력한 브러시리스 모터
- **21V 충전식 대포 송풍기** — 4000mAh 배터리 2개 포함
- **18V 30mm 전동가위** — 정밀 작업용
- **배터리 & 액세서리** — 2Ah/4Ah 배터리, 충전기 등

## 2. 캠페인 목적
- **매출 증대** (쿠팡 채널 중심)
- **매출 목표**: 월 1억원 (3개월 합계 3억원)

## 3. 타겟 오디언스
- **1차 타겟**: 40~50대 남성/여성
- **2차 타겟**: 50~60대 남성/여성
- **공통 특성 (추정)**:
  - 주말 DIY / 정원 관리에 관심
  - 가정용 전동공구 필요 (나무 전지, 낙엽 정리, 가지치기 등)
  - 가성비 중시하면서도 품질 신뢰 필요

## 4. 예산
- **월 1,000만원** (총 3개월 = 약 3,000만원)

## 5. 캠페인 기간

- 봄~초여름 시즌 (정원 관리 성수기)
