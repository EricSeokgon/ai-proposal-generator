## 1. 크리에이티브 컨셉 (3안)

### 컨셉 A: "검색의 정답" (Search Answer)

**대응 전략**: 전략 A — 봄 정원 시즌 검색 장악

| 항목 | 내용 |
|------|------|
| **빅 아이디어** | 쿠팡에서 "충전식 체인톱"을 검색하는 순간, 크래프트볼트가 가격-성능-리뷰 모든 면에서 "정답"으로 보이게 만든다 |
| **슬로건** | **"찾고 있던 그 공구, 크래프트볼트"** |
| **비주얼 무드** | 클린한 화이트 배경 + 오렌지 강조색. 제품 중앙 배치, 핵심 스펙 아이콘 배지. 모바일 썸네일에서 즉시 읽히는 정보 밀도 |

**핵심 카피 3종**

| 용도 | 카피 |
|------|------|
| 쿠팡 상품 제목 | `크래프트볼트 21V 충전식 체인톱 8인치 브러시리스 [배터리2개+충전기] 경량 1.2kg 정원관리` |
| 썸네일 텍스트 | `브러시리스 21V` / `디월트급 성능, 절반 가격` / `1년 무상 A/S` |
| 상세페이지 헤드라인 | `"30만원짜리 성능을, 반값에" — 브러시리스 모터 탑재 충전식 체인톱` |

### 컨셉 B: "주말의 프로" (Weekend Pro)

**대응 전략**: 전략 B — 유튜브 to 쿠팡, 콘텐츠 드리븐

| 항목 | 내용 |
|------|------|
| **빅 아이디어** | 평일엔 직장인, 주말엔 프로 정원사. 크래프트볼트를 손에 쥔 순간 "아마추어에서 프로로" 변신하는 순간을 담는다 |
| **슬로건** | **"주말엔 프로입니다 — 크래프트볼트"** |
| **비주얼 무드** | 실제 정원/마당 촬영 라이프스타일 영상. 봄 햇살 + 초록 잔디 + 오렌지 공구 색감 대비. Before/After 비교 숏 중심 |

**핵심 카피 3종**

| 용도 | 카피 |
|------|------|
| 쿠팡 상품 제목 | `[유튜브 화제] 크래프트볼트 21V 충전식 체인톱 브러시리스 경량설계 배터리2개 1년A/S` |
| 썸네일 텍스트 | `주말 정원관리 끝판왕` / `Before→After 3분` / `유튜버가 선택한 이유` |
| 상세페이지 헤드라인 | `"이 가격에 이 성능?" — 공구 유튜버 3인이 실사용 검증한 체인톱` |

### 컨셉 C: "봄 정원 올인원 키트" (Spring Garden Kit)

**대응 전략**: 전략 C — 정원관리 3종 세트 번들 프로모션

| 항목 | 내용 |
|------|------|
| **빅 아이디어** | 체인톱+송풍기+전동가위를 "정원관리 스타터 키트"로 패키징. "이것 하나면 우리 집 마당 끝"이라는 완결성 제공 |
| **슬로건** | **"마당이 있다면, 이 세트면 끝 — 크래프트볼트"** |
| **비주얼 무드** | 3개 제품의 프리미엄 패키지 언박싱 장면. 선물/기프트 느낌. 오렌지+블랙 통일 컬러. 가격 비교 인포그래픽 강조 |

**핵심 카피 3종**

| 용도 | 카피 |
|------|------|
| 쿠팡 상품 제목 | `[정원관리 3종세트] 크래프트볼트 충전식 체인톱+송풍기+전동가위 21V 배터리호환 봄시즌 특가` |
| 썸네일 텍스트 | `3종 세트 특가` / `개별구매 대비 30% 절약` / `배터리 호환 시스템` |
| 상세페이지 헤드라인 | `"체인톱, 송풍기, 전동가위 — 따로 사면 손해" 정원관리 올인원 세트` |

## 2. 메시지 체계

### 통합 메시지 체계

| 레벨 | 메시지 |
|------|--------|
| **Key Message** | 디월트급 성능, 절반 가격. 국내 A/S까지. 찾고 있던 바로 그 공구. |
| **Supporting 1** | 브러시리스 모터 탑재 — 디월트와 동일한 핵심 기술, 수명 2배 |
| **Supporting 2** | 21V 고출력 + 1.2kg 경량 — 누구나 한 손으로 가볍게 |
| **Supporting 3** | 1년 무상 A/S + 국내 배터리 수리 — 중국산과 다릅니다 |
| **Proof Point** | 브러시리스 모터 수명 50,000시간 / KC인증 / 배터리 2개 기본 포함 / 리뷰 평점 4.5+ |
| **RTB** | 가격표 직접 비교 (디월트 29만원 vs 크래프트볼트 14만원) + 핵심 스펙 동일 표기 |

## 3. 채널별 콘텐츠 플랜

### 3-1. 쿠팡 상품 리스팅 최적화

#### 상품 제목 전략

| 제품 | 최적화된 상품 제목 | 핵심 키워드 |
|------|------------------|-----------|
| 체인톱 | `크래프트볼트 21V 충전식 체인톱 8인치 브러시리스모터 경량1.2kg [배터리2개+충전기] 나무전지 정원관리 1년무상AS` | 충전식 체인톱, 전동 체인톱, 정원관리 |
| 송풍기 | `크래프트볼트 21V 충전식 대포 송풍기 브러시리스 [4000mAh배터리2개] 낙엽청소 정원관리 경량` | 충전식 송풍기, 낙엽 송풍기 |
| 전동가위 | `크래프트볼트 18V 충전식 전동가위 30mm 가지치기 원예가위 경량설계 여성사용OK 1년AS` | 충전식 전동가위, 가지치기 가위 |
| 3종 세트 | `[봄시즌특가] 크래프트볼트 정원관리 3종세트 체인톱+송풍기+전동가위 21V배터리호환 올인원` | 전동공구 세트, 정원관리 세트 |

#### 상세페이지 구성 (모바일 스크롤 순서)

| 순서 | 섹션 | 내용 |
|------|------|------|
| 1 | 히어로 배너 | "디월트급 성능, 절반 가격" + 제품 영웅샷 |
| 2 | 핵심 스펙 3줄 | 브러시리스 / 21V 고출력 / 1.2kg 경량 아이콘 |
| 3 | vs 비교표 | 크래프트볼트 vs 프리미엄 vs 중국산 3열 비교 |
| 4 | 사용 장면 | 정원 실사용 라이프스타일 사진 3~4장 |
| 5 | 상세 스펙 | 모터, 전압, 바 길이, 무게, 구성품 테이블 |
| 6 | 패키지 구성 | 본체+배터리2개+충전기 언박싱 Flat Lay |
| 7 | A/S 안심 보증 | 1년 무상 A/S + 국내 수리센터 |
| 8 | 고객 리뷰 하이라이트 | 실구매 리뷰 스크린샷 3~5개 |
| 9 | FAQ | 배터리 호환, 사용시간, 교환/반품, A/S |

### 3-2. 쿠팡 검색광고 키워드 & 카피

| 그룹 | 키워드 예시 | CPC 예상 | 전략 |
|------|-----------|---------|------|
| 헤드 | 충전식 체인톱, 전동 송풍기, 전동가위 | 400~600원 | 시즌 초 선점 (4월) |
| 미들 | 정원관리 전동공구, 가정용 체인톱 | 200~400원 | 안정적 운영 |
| 롱테일 | 충전식 체인톱 브러시리스, 경량 전동가위 여성용 | 100~200원 | 전환율 최적화 |
| 경쟁사 | 디월트 대안, 전동공구 가성비 추천 | 300~500원 | 비교 구매자 공략 |
| 시즌 | 봄 정원관리, 나무 전지 시기, 마당정리 | 150~300원 | 4~5월 한정 |

#### 입찰 시간대 전략

| 시간대 | 입찰 계수 | 이유 |
|--------|---------|------|
| 오전 6~9시 | 1.0x | 출근 전 탐색 |
| 오전 9~18시 | 0.8x | 비활성 시간 |
| **오후 18~21시** | **1.5x** | **골든타임** — 퇴근 후 쿠팡 피크 |
| 오후 21~24시 | 1.2x | 야간 구매 결정 |
| 주말 전체 | 1.3x | DIY/정원 관심 피크 |

## 4. 월별 크리에이티브 캘린더

### 4월: "검색 장악의 달" (컨셉 A 중심)

| 주차 | 활동 | 예산 |
|------|------|------|
| 1주 | 상품 리스팅 전면 리뉴얼 (제목/이미지/상세페이지) | 200만 (제작비) |
| 2주 | 검색광고 본격 운영 — 골든타임 입찰 강화 + A/B 테스트 | 300만 |
| 3주 | 키워드 최적화 + 리뷰 확보 활동 | 300만 |
| 4주 | 디스플레이 광고 추가 + 유튜버 섭외 착수 | 200만 |

**4월 예산**: 검색 600만 + 디스플레이 200만 + 제작 200만 = **1,000만원**

### 5월: "번들의 달" (컨셉 C 중심)

| 주차 | 활동 | 예산 |
|------|------|------|
| 1주 | 3종 세트 등록 + 세트 전용 상세페이지 | 300만 |
| 2주 | 세트 검색광고 + 유튜브 리뷰 1차 공개 | 250만 |

| 4주 | 성과 분석 및 6월 전략 조정 | 200만 |

**5월 예산**: 검색 500만 + 디스플레이 300만 + 제작 200만 = **1,000만원**

### 6월: "수확의 달" (A+C 병행)

| 주차 | 활동 | 예산 |
|------|------|------|
| 1주 | 고성과 키워드 집중 + "장마 전 마당정리" 메시지 전환 | 300만 |
| 2주 | 누적 리뷰 레버리지 + 세트 재프로모션 | 200만 |
| 3주 | 리타겟팅 강화 (5월 미구매자) + 유튜브 2차 콘텐츠 | 200만 |
| 4주 | 캠페인 종합 성과 분석 + 7월 전략 수립 | 300만 |

**6월 예산**: 검색 600만 + 디스플레이 300만 + 기타 100만 = **1,000만원**

## 5. 톤앤매너 가이드

| 요소 | 가이드 |
|------|--------|
| **보이스** | 신뢰감 + 친근함. "동네 형이 진짜 좋은 거 알려주는" 느낌 |

| **이미지** | 제품: 클린 화이트 스튜디오샷 / 라이프스타일: 자연광 실제 정원 |
| **타이포** | 고딕 계열, Bold~ExtraBold, 모바일 가독성 |
| **금지 표현** | "최저가", "업계 최초", "디월트보다 좋다", "완벽한/최고의" |
