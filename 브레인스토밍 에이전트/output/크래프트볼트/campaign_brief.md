## 2. 타겟 & 인사이트

### 타겟 페르소나 A: 김도현 (48세, 남성)

| 항목 | 내용 |
|------|------|
| 인구통계 | 48세 남성, IT 기업 부장, 연봉 6,500만원 |
| 가족 | 아내 + 고등학생 자녀 2명, 경기도 용인 단독주택 (작은 마당) |
| 심리그래픽 | 주말 정원사 자처, 가성비 중시하면서도 품질 타협 없음, "디월트 갖고 싶지만 가격이..." |
| 미디어 소비 | 유튜브(공구 리뷰) → 쿠팡 검색 → 네이버 가격비교 → 쿠팡 구매 |

| 구매 동기 | 봄이라 마당 나무 전지 필요. 5년 된 유선 드릴 교체 시기 |

| **Supporting 1** | 브러시리스 모터 탑재 — 프리미엄 브랜드와 동일한 핵심 기술 |

- **검색광고 55%**: 매출 직접 기여도 1위. 롱테일 40% 비중으로 CPC 최적화

- **콘텐츠 제작 10%**: 4월 상세페이지 리뉴얼 집중, 이후 점진적 축소
