## 2. 3C 분석

### Company — 크래프트볼트
| 강점 | 약점 |
|------|------|
| 디월트급 스펙(브러시리스, 21V) + 가성비 가격 | 낮은 브랜드 인지도 |
| 경량 설계 → 4050 여성/5060 타겟에 적합 | 쿠팡 내 리뷰 수 부족 (추정) |
| 1년 무상 A/S + 배터리 수리 서비스 | 제품 라인업 제한적 (체인톱, 송풍기, 전동가위) |
| 국내 브랜드 → 신뢰/소통 용이 | 오프라인 체험 채널 부재 |
| 오렌지+블랙 강렬한 브랜드 컬러 | 디월트 대비 글로벌 레퍼런스 부족 |

### Customer — 4050/5060 타겟
| 특성 | 4050 (1차 타겟) | 5060 (2차 타겟) |
|------|----------------|----------------|
| 쿠팡 이용도 | 높음 (로켓배송 헤비유저) | 급성장 중 (온라인 소비 주도 세대로 부상) |
| 구매 결정 요인 | 유튜브 리뷰 → 쿠팡 검색 → 가격비교 | 가족 추천(카톡) → 쿠팡 검색 → 리뷰 확인 |
| Pain Point | 디월트는 비싸고, 중국산은 불안 | 무겁고 복잡한 공구, A/S 불안 |
| 핵심 니즈 | 가성비 + 성능 보장 | 경량 + 쉬운 사용 + 국내 A/S |
| 구매 시즌 | 봄(4~5월) 정원관리 시작 시점 | 봄~여름 지속적 정원관리 |

### Competitor — 디월트 및 경쟁 구도
| 브랜드 | 포지셔닝 | 가격대 | 강점 | 약점 |
|--------|---------|--------|------|------|
| **디월트** | 프리미엄 프로 | 높음 (2~3배) | 글로벌 신뢰, 브랜드 파워 | 가정용 과잉 스펙, 고가 |
| **보쉬** | 프리미엄 가정용 | 중~높음 | DIY 라인업 강력 | 가격 부담 |
| **중국 브랜드** | 초저가 | 매우 낮음 | 가격 | A/S 불가, 품질 불안 |
| **크래프트볼트** | ⭐ 가성비 전문가급 | 중간 | 스펙+가격+A/S 균형 | 인지도 부족 |

> **시장 공백 발견**: "디월트급 스펙 + 중국산 가격대 + 국내 A/S" → 이 포지션이 비어 있음

## 3. 타겟 페르소나

### 페르소나 A: 김도현 (48세, 남성)

| 항목 | 내용 |
|------|------|
| 나이/성별 | 48세 남성 |
| 직업 | IT 기업 부장, 연봉 6,500만원 |
| 가족 | 아내 + 고등학생 자녀 2명 |
| 거주 | 경기도 용인 단독주택 (작은 마당 보유) |
| 라이프스타일 | 주말마다 마당 정리, DIY 가구 수리, 유튜브 공구 리뷰 즐겨봄 |
| 미디어 소비 | 유튜브(공구 리뷰, DIY 채널) → 쿠팡 검색 → 네이버 가격비교 |
| 현재 공구 | 5년 된 유선 전동드릴 + 수동 톱 |
| **Pain Point** | "디월트 사고 싶은데 체인톱 하나에 30만원은 부담... 중국산은 배터리 터질까 무섭고" |
| **구매 동기** | 봄이라 마당 나무 전지 해야 함. 충전식으로 바꾸고 싶음 |
| **구매 결정 기준** | ① 가격 대비 성능 ② 리뷰 평점 4.5 이상 ③ 로켓배송 ④ A/S 가능 여부 |

### 페르소나 B: 박순영 (57세, 여성)

| 항목 | 내용 |
|------|------|
| 나이/성별 | 57세 여성 |
| 직업 | 은퇴 후 텃밭 가꾸기, 전원생활 입문 |
| 가족 | 남편(60세) + 출가한 성인 자녀 |
| 거주 | 충남 아산 전원주택 (넓은 마당 + 텃밭) |
| 라이프스타일 | 매일 텃밭 관리, 주말 마당 정리, 카톡으로 정보 공유 |
| 미디어 소비 | 카카오톡(가족/이웃 추천) → 쿠팡 앱 → 상품 리뷰 사진 중심 |
| 현재 공구 | 수동 전지가위 + 빗자루 |
| **Pain Point** | "전지가위로 나뭇가지 자르면 손목이 아파... 전동으로 바꾸고 싶은데 무거우면 못 쓸 것 같고" |
| **구매 동기** | 봄 나무 전지 + 여름 풀 정리에 전동가위/송풍기 필요 |
| **구매 결정 기준** | ① 가벼운 무게 ② 사용 쉬움 ③ 국내 A/S ④ 가족이 추천 ⑤ 리뷰 사진 좋음 |

## 4. 전략 방향 3개

### 전략 A: "봄 정원 시즌 검색 장악" 🔍

| 항목 | 내용 |
|------|------|
| **핵심 콘셉트** | 쿠팡 검색광고 집중 투자로 "충전식 체인톱", "전동 송풍기" 등 핵심 키워드 상위 노출 |
| **타겟 세그먼트** | 쿠팡에서 전동공구를 검색하는 구매 의향 높은 4050 남성 |
| **포지셔닝** | "디월트 성능, 절반 가격" — 검색 결과에서 가격+리뷰로 즉시 전환 |
| **예산 배분** | 검색광고 70% + 상품 리스팅 최적화 20% + 리뷰 관리 10% |
| **예상 효과** | 즉각적 매출 전환, ROAS 300~500% 목표 |
| **리스크** | 브랜드 구축 효과 제한, 키워드 경쟁 심화 시 CPC 상승 |
| **적합 시기** | 4월 (시즌 초기 선점) |

### 전략 B: "유튜브 to 쿠팡" 콘텐츠 드리븐 📹

| 항목 | 내용 |
|------|------|
| **핵심 콘셉트** | 유튜브 공구 리뷰어/DIY 크리에이터 협업으로 "크래프트볼트 vs 디월트" 비교 콘텐츠 제작 → 쿠팡 유입 |
| **타겟 세그먼트** | 유튜브 공구 리뷰를 시청하는 4050 남성 + 카톡 공유로 유입되는 5060 |
| **포지셔닝** | "실사용 검증된 가성비 전동공구" — 리뷰어의 입을 통한 간접 보증 |
| **예산 배분** | 유튜브 인플루언서 40% + 쿠팡 검색광고 40% + 콘텐츠 제작 20% |
| **예상 효과** | 브랜드 인지도 상승 + 검색량 증가 + 장기 자산(영상 콘텐츠) |
| **리스크** | 초기 매출 효과 느림, 인플루언서 섭외 리드타임 |
| **적합 시기** | 4월 콘텐츠 제작 → 5~6월 본격 효과 |

### 전략 C: "정원관리 3종 세트" 번들 프로모션 🎁

| 항목 | 내용 |
|------|------|
| **핵심 콘셉트** | 체인톱+송풍기+전동가위 번들 세트를 시즌 한정 특가로 구성, 객단가 극대화 |
| **타겟 세그먼트** | 전원생활 입문 5060 + 마당 보유 4050 가정 |
| **포지셔닝** | "정원관리 올인원 솔루션" — 개별 구매보다 세트가 압도적으로 합리적 |
| **예산 배분** | 쿠팡 디스플레이 광고 40% + 검색광고 30% + 할인 프로모션 비용 30% |
| **예상 효과** | 객단가 2~3배 상승, 배터리 호환 생태계 락인(Lock-in) |
| **리스크** | 마진율 하락, 재고 관리 복잡성 |
| **적합 시기** | 5월 (시즌 중반, 구매 확신 시점) |
